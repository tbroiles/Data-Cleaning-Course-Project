# Data Code Book

## Variable Annotation

### Variable Prefix

* “t” - Time series data
* “f” - Fast Fourier Transform of the time series data.

### Variable types

* "Activity" - Physical activity the user performed
* “BodyAcc” - Linear acceleration
* “GravityAcc” - Linear acceleration due to gravity
* “BodyAccJerk” - Time derivative of linear acceleration
* “BodyGyro” - Rotational acceleration
* “BodyGyroJerk” - Time derivative of rotational acceleration

### Statistical operation on each variable

* “mean()” - Mean
* “std()” - Standard Deviation

### Variable dimensions

* “X” - operation in the phone’s X dimension
* “Y” - operation in the phone’s Y dimension
* “Z” - operation in the phone’s Z dimension
* “Mag” - operation on the magnitude of phone’s dimensions

### Activity Types

* "walking"
* "walking upstairs"
* "walking downstairs"
* "sitting"
* "standing"
* "laying"
